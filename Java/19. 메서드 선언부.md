### 매개변수(parameter) 선언
매개변수?
> 매서드가 작업수행에 필요한 값들 제공받을 수 있다.

1. 각 변수 간의 **구분은 쉼표**를 사용

2. 두 개 이상의 **변수의 타입이 같아도 변수의 타입 생략 X**

```java
int hamsu(int x, int y, int z // 쉼표로 분리, 타입 생략 X){ }
```

### 반환타입
>메서드의 작업수행 결과인 반환값의 타입을 작성

#### 반환값이 없다면?
`void` 작성

```java
 void letMeSleep(){System.out.println("I'm gonna die!");}
 // 반환값이 없어 void 작성
```
