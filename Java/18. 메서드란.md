### 메서드가 나눠진 부분

```java
반환타입 메서드이름 (타입 변수명, 타입변수명, ...) // 선언부
{
	// 메서드 호출시 수행될 코드 // 구현부
}

int add(int a, int b) // 선언부
{
	int result = a + b; //구현부
    return result;
}
```
